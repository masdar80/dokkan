# 🎭 حالات الاستخدام (Use Cases)

يصف هذا المستند التفاعلات بين المستخدم (صاحب الدكان) والنظام.

## 👥 الجهات الفاعلة (Actors)
- **صاحب الدكان (Owner):** المستخدم الأساسي الذي يدير المخزون، المبيعات، المشتريات، والتقارير.

---

## 📋 قائمة حالات الاستخدام

### 1. إدارة المواد (Manage Items)
- **الوصف:** إضافة مادة جديدة أو تحديث بياناتها.
- **الهدف:** الحفاظ على سجل دقيق للمواد المتاحة.

### 2. تسجيل عملية بيع (Record Sale)
- **الوصف:** اختيار مادة، تحديد الكمية، وتأكيد البيع مع تسجيل سعر الصرف.
- **الهدف:** توثيق الدخل وتحديث المخزون.

### 3. تسجيل عملية شراء (Record Purchase)
- **الوصف:** إضافة كميات جديدة من مادة معينة مع تحديد سعر الشراء وسعر الصرف.
- **الهدف:** زيادة المخزون وتوثيق التكاليف بالدولار.

### 4. تحديث سعر الصرف (Update Exchange Rate)
- **الوصف:** جلب السعر من الإنترنت أو إدخاله يدوياً والموافقة عليه.
- **الهدف:** ضمان دقة حسابات الأرباح بناءً على سعر السوق الحالي.

### 5. استعراض التقارير (View Reports)
